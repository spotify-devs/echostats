# Welcome to EchoStats 🎵

EchoStats is a **self-hosted** web application that connects to your Spotify account, tracks your listening history, and provides beautiful analytics and visualizations about your music taste.

## Key Features

- **🔐 Spotify OAuth** — Secure authentication via Spotify
- **📊 Rich Analytics** — Listening time, top artists, tracks, genres, patterns
- **📈 Interactive Charts** — Line, bar, pie, heatmap, and radar charts
- **📋 Data Tables** — Sortable, filterable tables with pagination
- **🎯 Recommendations** — Discover new music based on your taste
- **🎵 Playlist Management** — Create and manage playlists
- **📱 PWA** — Install as an app on any device
- **🏠 Self-Hosted** — Your data stays on your server
- **📚 Full History Import** — Import Spotify privacy data exports

## Tech Stack

| Component | Technology |
|-----------|-----------|
| Backend | Python 3.12 / FastAPI |
| Frontend | Next.js 16 / React 19 |
| Database | MongoDB 7 |
| Cache | Redis 7 |
| Worker | ARQ (async task queue) |

## How It Works

1. **Authenticate** with your Spotify account
2. **EchoStats syncs** your listening data every 15 minutes
3. **Explore** your analytics through the dashboard
4. **Import** historical data from Spotify privacy exports for full history
