# Welcome to EchoStats 🎵

EchoStats is a **self-hosted** Spotify analytics platform that connects to your Spotify account, automatically syncs your listening history, and delivers 40+ pages of beautiful dashboards, charts, and insights about your music taste. Think of it as **Spotify Wrapped — but year-round**, running entirely on your own server.

## Who is EchoStats for?

- **Self-hosters** who want full control over their data and infrastructure
- **Privacy-conscious Spotify users** who don't want analytics data sent to third parties
- **Music data enthusiasts** who want deeper insights than Spotify provides natively
- **Developers** looking for a modern full-stack reference project (FastAPI + Next.js + MongoDB)

## Key Features

### 🎛️ Analytics & Visualization
- **40+ Dashboard Pages** — Stats, charts, heatmaps, radar charts, listening calendars, mood analysis, and more
- **Listening Analytics** — Top tracks, artists, genres, and albums with time-range filters (week / month / year / all-time)
- **Music DNA** — Audio feature radar showing danceability, energy, valence, acousticness, tempo, and more
- **Artist Map** — Explore connections between favorite artists with play-share distribution
- **Taste Profile** — Deep analysis of your listening patterns and preferences
- **Year in Review** — Spotify Wrapped-style annual summary generated from your data

### 🔄 Sync & Data
- **Auto Sync** — Background worker syncs your Spotify data every 15 minutes via ARQ cron
- **Data Import** — Import your full Spotify extended streaming history (JSON from privacy data export)
- **Sync Job Tracking** — View step-by-step sync progress with detailed API call history
- **API Logs Dashboard** — Monitor all Spotify API calls with status distribution and latency stats

### 🎵 Playback & Discovery
- **Spotify Playback Control** — Play, pause, skip, view queue, and switch devices from the dashboard
- **Recommendations** — Discover new music based on your listening patterns
- **Browse** — New releases, featured playlists, and categories from Spotify

### 🎨 Customization & UX
- **6 Themes + 8 Accent Colors** — Dark, Light, Dim, Ocean, Midnight, Forest + custom color picker
- **PWA** — Install on any device (mobile and desktop), works offline
- **Single-User Auto-Login** — Auto-authenticates when only one user exists (ideal for self-hosting)
- **Update Notifications** — Dashboard banner when a new version is available on GitHub

### 🏠 Deployment
- **Docker Compose** — One-command deployment with all services included
- **Helm Chart** — Kubernetes deployment via OCI registry with full configurability
- **Version Indicator** — Deployed version shown in sidebar and API health endpoint

## How It Works

1. **Authenticate** — Connect your Spotify account via OAuth
2. **Sync** — EchoStats automatically syncs your listening data every 15 minutes
3. **Explore** — Browse 40+ dashboard pages of analytics, charts, and insights
4. **Import** — Optionally import your full Spotify history from privacy data exports

## Tech Stack

| Layer | Technology | Purpose |
|-------|-----------|---------|
| Frontend | Next.js 16 / React 19 / TypeScript | SSR dashboard with 42 pages |
| Styling | Tailwind CSS 4 / Framer Motion | 6 themes + spring animations |
| Charts | Recharts | Interactive data visualizations |
| Backend | Python 3.12 / FastAPI | REST API with 30+ endpoints |
| Database | MongoDB 7 / Beanie ODM | Document storage for analytics |
| Cache / Queue | Redis 7 / ARQ | Background sync + task queue |
| Container | Docker / Helm | Kubernetes-ready deployment |
| CI/CD | GitHub Actions | Automated testing + GHCR publishing |
